Les balises d'information

Les balises d'information, ça veut dire quoi ?

Les balises d'information sont indispensables à la construction d'une page web. Elle permet au navigateur d'avoir toutes les informations dont il a besoin afin d'intéprêter correctement le programme que nous avons créés.

<!DOCTYPE HTML> : la base

Cette balise est spéciale par sa forme et sa position. On la dispose à la première ligne du code afin de dire au navigateur que notre page est programmé en HTML5.

Pareillement, on va intégrer toute notre page web dans la balise <html>.
On procède de la façon suivante :

<html>
contenu de la page web
<html/>

<head>: les informations indispensables

Ici, nous allons mettre toutes les informations que le navigateur aura besoin pour comprendre et intéprêter notre page web. On place la balise <head> juste en dessous de la balise <html>. On intègre dans cette balise les balises d'information suivantes :

<meta>

Les balises META servent à placer des métadonnées dans une page HTML, ces métadonnées ne seront pas affichées sur la page.

<title>

Cette balise permet de donner un titre à la page web, il sera visible sur la barre d'onglet du navigateur

<script>

Cette balise permet d'ajouter un script à la page web

<style>

Cette balise permet d'ajouter une feuille de style CSS à la page web. Cette feuille de style mettra en forme la page web

<body>

La balise body indique au navigateur le corps de la page

<header>

Cette balise permet de dire au navigateur que tout ce qui est dans <header> est en haut de la page

Autre

<details> et ou <summary>

Cet élément permet de baliser un contenu quelconque (texte, image, listes, tableaux, formulaires, etc) pour apporter une information sur celui ci.

<footer>: le pied de page

La balise <footer> permet de dire au navigateur que tout ce qui se trouvera dans cette balise sera situé tout en bas de la page web. On peut y trouver des informations de contact, de licence...
