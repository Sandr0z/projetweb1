Structurer le contenu de la page web

Ici, nous allons voir toutes les balises permettant de structurer l'information sur la page web. Comment mettre un titre, puis un sous-titre; faire un retour à la ligne ; afficher un menu ; une citation...

<h1;2;3;4;5;6....> Niveau de mon titre </h>

La balise <h> permet de structurer ses titres dans la page. Plus le nombre est petit, plus le titre sera haut hiérarchiquement.

Exemple :
<h1> Mon titre le plus grand </h1>
<h3> Mon premier sous titre </h2>

<p> Mon paragraphe </p>

Cette balise est utilisée pour définir que le texte s'y trouvant à l'interieur est un paragraphe.

Exemple :
<p> Voici mon paragraphe sur la communication non-violente au quotidien mise en avant par Marshall Rosenberg </p>

<br/> :
Les retours à la ligne

Cette balise autofermante permet de faire un retour à la ligne

Exemple : <p>Voici mon paragraphe une fois de plus sur la communication non-violente au quotidien. <br/>
Elle a été théorisée par l'enseignant chercheur Marshall Rosenberg.</p>

<hr> : les ruptures thématiques

Cette balise permet de faire une rupture thématique sur du texte, certains navigateurs tracent une ligne horizontale.

Exemple :
<p>Voici mon paragraphe encore une fois sur la communication non-violente au quotidien. <hr/>

Elle a été théorisée par l'enseignant chercheur Marshall Rosenberg.</p>

<blockquote> : de grandes citations

Cette balise permet de montrer au navigateur que le texte suivant est une grande citation.

Exemple : <blockquote> Ma grande citation </blockquote>
Ce qui donne ça :
Oui, je peux accepter que tu me dises
Ce que j’ai fait ou pas fait.
Et je peux accepter tes interprétations.
Mais je t’en prie ne mélange pas les deux.

<q> : Les citations courtes

Cette balise permet, comme celle juste au dessus, d'indiquer au navigateur que le texte suivant est une citation. Cependant, on utilise la balise <q> plutôt pour les petites citations.

Exemple : <q> Ma petite citation </q>
Ce qui donne ça :
Un grand pouvoir implique de grandes responsabilités.

<table> : Les tableaux

Cette balise permet d'afficher un tableau

Exemple : <table>Mes données de tableaux</table>

<article> : Le contenu autonome

Cette balise permet d'ajouter un élément autonome à la page web

Exemple : <article class="la_description_de_notre_element"> Notre élément </article>

<menu> : Ajouter un menu sur sa page HTML5

Cette balise permet d'ajourer un menu sur sa page HTML5, elle est cependant obsolète actuellement

Exemple : <menu type="context"> Nos éléments de menu </menu>

<section> : Un contenu générique

Cette balise permet de définir une section générique sur un document

Exemple :
<section>
<h1>Titre</h1>
<p>paragraphe en lien avec la section</p>
</section>

<aside> : "Aparté" en anglais

Cette balise permet de définir une partie du document qui n'aura qu'un rapport indirect avec le contenu du document.

Exemple :
<body>
<aside>Mon texte ayant une mise en forme alternative</aside>
<p>Mon texte normal</p>
</body>

<figure> : Balise de schéma

Cette balise permet d'importer un schéma dans une page HTML5, elle peut être légendée avec la balise <figcaption>

Exemple : <figure><img src="../images/img.png"><figcaption>Une photo de moi</figcaption> </figure>


<nav> : Les liens de navigation

Cette balise permet de définir une section de page ayant des liens vers d'autres pages, fragments de pages ou documents liés au site.

Exemple :
<nav>
<ol>
<li>Lien 1 vers une page du site</li>
<li>Lien 2 vers un fragment de la page actuelle</li>
</ol>
</nav>

<div> : Conteneur indépendant

Cette balise permet d'attribuer un conteneur générique à l'interieur de la page. Il peut être mis en forme indépendamment de la page web grâce à du code CSS5.

Exemple :
<body>
<p>Un élément du body</p>
<div class"le_nom_du_conteneur">Mon conteneur indépendant</div>
</body>

<span> : Conteneur indépendant "inline"

Comme la balise <div> , la balise <span> sert à définir un conteneur indépendant de mise en forme générale. Celle ci va cependant être utilisée sur du texte, contrairement à <div> qui s'applique à n'importe quel éléments d'une page.

Exemple :
<body>
<p>Un élément du body</p>
<span class"le_nom_du_conteneur">Mon conteneur textuel</span>
</body>

<cite> : La citation en évidence

Cette balise permet de mettre en évidence une citation

Exemple : <p><cite> Banlieusard et fier de l'être, on n'est pas condamnés à l'échec.</cite> -Kery James</p>
=

Banlieusard et fier de l'être, on n'est pas condamnés à l'échec. -Kery James

<canvas> : L'éléments de canevas graphiques

La balise <canvas> permet de modifier une zone graphique via un script

<objects> : Une ressource externe dans la page

Cette balise permet d'ajouter une ressource externe sur la page web, comme un document PDF, ou une grande image.

Exemple : <object type="type_de_fichier" data="lien_du_fichier"> blablabla </object>

<time> : représenter un format temporel

Cette balise permet de représenter une date, une heure ou bien un jour dans la page web.

Exemple : <time datetime=20:00>20h</time>, l'heure de la signature du decret

<meter> ou <progress>: Des intervalle scalaire

Ces balises représentent une valeur scalaire dans un intervalle donné ou une valeur fractionnaire. Elles sont généralement affichées comme des barres de progression.

Exemple : <meter id="fuel" min="0" max="100" low="33" high="66" optimum="80" value="50"> at 50/100 </meter>

= at 50/100

<abbr> : Les abbreviations

Cette balise permet de définir des abreviations ou des acronymes.

Exemple : On peut utiliser le <abbr title="Cascading Style Sheets">CSS</abbr> pour styliser sa page web.

= : On peut utiliser le CSS pour styliser notre page web.

<adress> : Les éléments de contact

Cette balise permet d'indiquer des informations de contact comme l'adresse, le téléphone ou le mail d'une personne ou d'une organisation.

Exemple : <address><a href="tel:+33777951690">Téléphone</a></address>

=
Téléphone

<dfn> : Un terme en train d'être définit

Cette balise est utilisée afin d'indiquer le terme qui est en train d'être défini dans une phrase ou un paragraphe.

Exemple : <dfn id="def-outrecuidance">Une outrecuidance</dfn> est une figure de style mettant en avant l'éxageration.

= Une outrecuidance est une figure de style mettant en avant l'éxageration.

<code> : L'élément de code sur une page web.

Cette balise permet d'afficher des caractères comme un fragment de code machine

Exemple : La commande <code>git commit- m "description-de-sa-version"</code> permet d'ajouter une description à une nouvelle version de projet sur github avec git.

= La commande git commit -m "description-de-sa-version" permet d'ajouter une description à une nouvelle version de projet sur github avec git.

<kbd> : l'élément de saisie clavier

Cette balise permet d'ajouter des saisies clavier au texte, afin de mettre en avant un raccourci clavier ou une suite de combinaison.

Exemple : Faites <kbd>Ctrl</kbd> + <kbd>F</kbd> pour rechercher un élément dans la page.

= Faites Ctrl + F pour rechercher un élément dans la page.

<var> : Les éléments de variables

Cette balise permet de mettre en avant une ou plusieurs variables, utiles lors d'une explication mathématique.

Exemple : La variable <var>a</var> n'est pas égale à la variable <var>b</var>

= La variable a n'est pas égale à la variable b.

<ins> : Le texte ajouté sur un document

Cette balise représente du texte ajouté à un document par un utilisateur tier, utile notamment pour les pad collaboratifs comme Framapad ou Google Docs.

Exemple : Ceci e<ins>s</ins>t une faute d'orthographe.

= Ceci est une faute d'orthographe.

<del> : Un texte supprimé

Cette balise permet d'afficher du texte qui a été supprimé par un utilisateur tier, de la même manière que la balise juste au dessus.

Exemple : Bugs connus -> <del>Problème de bluetooth sur la rom custom</del><ins> fixed!</ins>

= Bugs connus -> Problème de bluetooth sur la rom custom fixed!

<b> : Un contenu attirant l'attention

Le but de cette balise est de mettre en évidence une partie de texte importante. Elle la met automatiquement en gras, alors il faut faire attention à son utilisation et ne pas trop en abuser.

Exemple : On ne jette <b>PAS</b> ses mégots par terre.

= On ne jette PAS ses mégots par terre.

<strong> : L'élément de haute importance

Même utilité que la balise <b>, mais un cran au dessus. Son information sera priorisée par le navigateur et les moteurs de recherche. L'effet visuel reste le même tant qu'il n'est pas mis en forme.

<u> : L'éléments souligné (je ne conseille pas de l'utiliser)

Cette balise permetde souligner un texte, elle est cepepdant obsolète. Il faut maintenant utiliser la balise CSS3 underline afin de "décorer" du texte en le soulignant par exemple.

<i> : Morceau de texte différencié

Cette balise permet de différencer un morceau de texte dans un paragraphe. Le texte est nottament interprété par défaut en italique.

Exemple : <p>Il faut toujours séparer les <i>intépretations</i> et les <i>faits</i> lorsqu'on applique la communication non-violente au quotidien</p>

= Il faut toujours séparer les intépretations et les faits lorsqu'on applique la communication non-violente au quotidien

<em> : Éléments de mise en emphase

Cette balise, comme la précédente, permet de mettre en emphase un morceau du texte

Exemple : <p>Il faut toujours séparer les <em>intépretations</em> et les <em>faits</em> lorsqu'on applique la communication non-violente au quotidien</p>

= Il faut toujours séparer les intépretations et les faits lorsqu'on applique la communication non-violente au quotidien

<mark> : Du texte marqué

Cette balise permet de marquer une partie de texte, comme on surlignerai un texte sur un papier avec un stabilo.

Exemple : L'azote liquide est à une température de <mark>-195.79°C</mark>

= L'azote liquide est à une température de -195.79°C.

<small> : Des éléments plus petits

Cette balise permet de mettre en forme une partie de texte en plus petit. Elle peut être utilisé en bas de page par exemple pour les mentions légales, termes d'un contrats..... Je déconseille l'utilisation de cette balise étant donné que les possibilités de mises en formes sont limitées, contrairement aux utilisations possibles avec le langage CSS3.

Exemple :
<small>Cette page a été écrite en 2019 par un grand monsieur. Licence open source.</small>

Exemple :
Cette page a été écrite en 2019 par un grand monsieur. Licence open source.

<sup> / <sub> : Exposant et souscriptions

Ces balises permettent d'affecter des exposants ou des souscriptions à des nombres, indispensables en mathématique ou en chimie.

Exemple : La molécule d'eau est H<sub>2</sub>O / A*A = A<sup>2</sup>.

= La molécule d'eau est H2O / A*A = A2.

Les listes classiques et ordonnées

Il y a plusieurs types de listes. Il y a les listes ordonnées qu'on met en utilisation avec la balise <ol> et les listes classiques avec <ul>. Pour ajouter un élément à la liste en écriture, il faut utiliser la balise <li> pour les deux types de listes.

Exemple : <ol><li>Premier élément de liste ordonnée</li><li>Deuxième élément de liste ordonnée</li></ol>
<ul><li>Premier élément de liste normale</li><li>Deuxieme élément de liste normale</li></ul>

=
  1. Premier élément de liste ordonnée
  2. Deuxième élément de liste ordonnée
  • Premier élément de liste classique
  • Deuxième élément de liste classique

<dl> : Les listes descriptives

Cette balise permet de faire une liste descriptive. On ajoute des éléments à cette liste avec les balises <dt> pour les éléments de titre, et la balise <dd> pour la description de titre dans la liste.

Exemple : <dl><dt>Nouveauté 1.1</dt><dd>Optimisation de la batterie</dd><dd>Compatible Android 10</dd>

=
Nouveauté 1.1
Optimisation de la batterie
Compatible Android 10

<tbody> : Corps de tableau

Cette balise permet de regrouper un ou plusieurs éléments à mettre en forme dans le corps d'un tableau. On peut le titrer avec la balise <caption>. Pareillement, les items du tableau sont mis en forme avec les balises <tr> <th> et <td>. Les types d'item sont mis en forme avec la balise <thread>.

Exemple :

Voyage à Marrakech
Bien à réflechir Argent à prévoir ~
Voyage avion A/R 170€
Se nourrir sur place 10€/j
Se loger sur place 15€/nuit
Sous bonus à prévoir 200€
