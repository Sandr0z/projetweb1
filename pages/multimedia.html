Les balises multimedia, à quoi servent t-elles ?

Ces balises servent à mettre en avant un contenu multimedia à l'interieur d'une page web. Ça peut être un morceau de musique au format MP3, comme une vidéo, une photo, etc.

Les différentes balises

<img/>, l'affichage d'un fichier image.

Cette balise permet d'afficher une image à l'interieur de la page web.

Exemple : <img src="liendemonimage.png"/> =

<audio> ; <video> ; <source>

Ces balises sont un peu spéciales, ne fonctionnent pas comme les autres. La balise <audio> sert à mettre un fichier audio sur la page web, et la balise <video> sert à mettre un fichier vidéo sur la page web. Seulement, on ne va pas intégrer le lien de ces fichiers dans les balises <audio> ou <video>, c'est là qu'on utilisera la balise <source>. Elle servira à sourcer le contenu multimedia audio ou vidéo pour le navigateur.

Exemples :
<video> <source src="lelienversmavideo"/> </video>
ou
<audio> <source src="lienversmonfichieraudio"/> </audio>

<a>, les liens hypermédia

Cette balise permet de rediriger n'importe quel élément de la page web vers un élément externe. Cette balise peut fonctionner sur une image, une vidéo, du texte. Avant, on apppelait ça des liens hypetextes, maintenant on préfère utiliser le terme hypermedia car nous pouvons cliquer sur d'autres médias que sur du texte pour se rediriger vers un lien externe.

Exemple : <a href="url-de-ma-redirection"> mon texte (ou media) cliquable </a>
